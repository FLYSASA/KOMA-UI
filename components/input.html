# Input 输入框

Attributes

组件将被渲染为 input 标签，因此你可以使用所有的 input 原生属性。

参数 说明 类型 可选值 默认值
placeholder 原生属性 String —— ——
type 原生属性(当type为textarea时渲染为文本输入框) String —— ——
readonly 是否只读 Boolean true，false false
disabled 禁用状态 Boolean true，false false
error error提示 String —— ——
